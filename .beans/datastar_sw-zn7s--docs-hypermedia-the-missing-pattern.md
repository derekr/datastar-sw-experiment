Core topic. Hypermedia as the Engine of Application State (HATEOAS). What it means to build 'hypermedia-driven' applications. Links, forms, and the 'follow your nose' principle. How Datastar's server-driven approach embraces hypermedia — the server sends HTML with embedded links and forms, the client just follows them. Contrast with REST APIs that return JSON and require out-of-band knowledge.
